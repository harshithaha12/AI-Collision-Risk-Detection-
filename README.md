# AI-Collision-Risk-Detection

A smart vehicle safety project that predicts possible collisions between vehicles by analyzing factors like speed, rain, fog, and road visibility, and sends alerts when collision risk increases.

## Features

* Collision risk prediction
* Weather-based analysis
* Fog and rain condition simulation
* Adjustable vehicle speed inputs
* Real-time risk alerts

## Tech Stack

* Python
* HTML
* CSS
* JavaScript
* Machine Learning

## Project Structure

* `backend/` → ML model and Flask backend
* `frontend/` → User interface

## Future Improvements

* Real-time sensor integration
* Live traffic data
* Advanced AI prediction models
